# 🍔 MERN Food Delivery App

A full-stack food ordering platform built with the MERN stack.

## 🚀 Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Tailwind CSS
- Context API + useReducer

## ✨ Features
- User authentication
- Browse food items
- Add to cart
- Order management
- Responsive UI
- RESTful APIs
